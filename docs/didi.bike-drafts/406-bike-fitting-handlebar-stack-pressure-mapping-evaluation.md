# Precision Wrenching: Torque Calibration and Saddle Pressure Mapping

## Step 1: Spacers Installation and Torque Tolerance
First, secure the bicycle in a professional repair stand. Clean the fork steerer tube thoroughly. When inserting carbon spacers beneath the stem to raise handlebar stack, inspect the stack for play/slop detection. Use a calibrated torque wrench to secure the stem pinch bolts. Respect the specified torque tolerance of 5.0 Nm. Apply medium-strength thread lock to prevent bolt backing under road vibrations. Never exceed recommended torque values on carbon components. Riders vary. Our mechanical checklist shows that individual cockpit tolerance varies, meaning that standardized frames fail to prevent thread damage.
