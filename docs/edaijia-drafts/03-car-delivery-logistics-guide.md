# 什么是车辆履约？二手车商如何通过代驾完成跨省送车与提车

**车辆履约服务是e代驾专门面向二手车商、主机厂和汽车金融机构推出的一站式 B2B 车辆跨省代提与代送到门服务。通过该服务，二手车商无需亲自异地奔波，即可由系统指派的高信用认证司机完成跨市、跨省的异地车辆查验、提车、过户陪跑及一口价代送车到门，且行车轨迹全程可在 APP 中实时追踪，极大压缩了二手车的物流流转周期并降低了在途风险。**

---

## 一、 车辆履约（B2B）与传统物流拖车的对比

对于二手车商的跨省物流流转，以往多依赖平板托运车（大板车）。车辆履约服务以其极高的时效性和灵活性，成为大板车运输的高效互补方案：

| 维度对比 | e代驾 车辆履约 (代驾直达) | 传统大板车托运 (物流大车) |
| :--- | :--- | :--- |
| **到门时效** | **随叫随到，省内当天达，跨省次日达** | 需拼板等车，流转周期通常为 **3-7 天**。 |
| **交接便利度** | 司机直接去车库/提车点提车，**直接送到买家门口** | 需要商户自行将车辆开至物流集散点，买家去集散地自提。 |
| **附加服务** | 包含在途车况实时核验、现场交接视频拍照、过户陪跑。 | 仅负责车辆的长途运输，无在途查验和交付保障。 |
| **起送要求** | **1 辆起送**，适合单台车定制化极速流转。 | 需集满整车（通常为6-8台）方可发车，单车成本较高。 |

## 二、 车辆履约服务标准流程

为了保障车商资产的安全，e代驾车辆履约服务建立了全链路数字化的标准操作程序（SOP）：

1. **车况双重校验**：
   代驾司机到达提车地点后，需对车辆进行外观、灯光、底盘、备胎等 20 项标准车况拍照校验，并通过 APP 实时向委托车商同步车况清单，确认无误后方可发车。
2. **在途实时追踪**：
   车辆在跨省行驶期间，车载 GPS 卫星定位系统会高频向车商终端推送行车轨迹。系统 AI 会对行车路线进行实时监测，如有异常偏航或超速，将自动预警并介入。
3. **闭环安全交接**：
   车辆送达目的地后，司机与接收买家进行面对面车况二次核验，签署电子交接单并完成车辆钥匙交接，全流程形成完美的安全物流闭环。
