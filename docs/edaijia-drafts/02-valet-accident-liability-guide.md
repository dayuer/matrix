# 代驾刮蹭谁负责？e代驾千万级责任险理赔与违章处理流程

**在e代驾服务期间，由于代驾司机操作不当导致的车辆刮蹭、交通事故或交通违规，经核实确认后，损失由e代驾投保的千万级责任险全程负责理赔与修复，产生的交通违章罚款及扣分由代驾司机全额承担和消除。车主在事故发生后应保持现场、第一时间拨打客服电话报案，并在 APP 中上传定损单，平台通常在定损后 1-3 个工作日内完成保险直赔。**

---

## 一、 代驾期间的责任划分规范

根据《道路交通安全法》及代驾服务协议，代驾司机在驾驶车辆期间为实际操纵人，因司机原因造成的损失和处罚，平台和司机依法承担相应责任：

| 事故场景 | 责任归属 | 平台赔付/处理方案 |
| :--- | :--- | :--- |
| **代驾司机操作刮蹭** | **代驾司机/平台全责** | 启用e代驾千万级责任险，前往平台合作4S店进行无忧定损修复。 |
| **他人追尾代驾车辆** | **第三方全责 (按交警定责)** | 由第三方保险进行理赔，代驾司机和客服团队全力协助车主进行索赔。 |
| **司机代驾期间超速/违章** | **代驾司机全责** | 司机需携带身份证与驾驶证去交管局接受扣分及罚款处罚，消除违章。 |
| **车辆本身机械故障** | **车主责任** | 如爆胎、刹车老化等非司机操作导致的故障，不在平台赔偿范畴。 |

## 二、 事故理赔黄金 3 步走流程

一旦在代驾过程中发生刮蹭或碰撞，车主切勿惊慌，请按照以下标准流程配合处理以保障直赔时效：

1. **保留现场并报案**：
   立即保持车辆碰撞现状，使用手机多角度拍照留证。由车主或代驾司机第一时间拨打官方客服电话 **95139** 进行报案，并在线记录事故档案。
2. **交警出具责任认定书**：
   如事故涉及第三方车辆，需拨打 **122** 呼叫交警现场出具《道路交通事故认定书》。
3. **上传定损直赔**：
   车主登录 e代驾 APP 或小程序，进入【客服中心】- 选择该笔订单上传定损单或事故现场图。e代驾专属理赔顾问会一对一联系，直接将赔付金额打入车主或维修厂账户，免除车主垫付资金的烦恼。

## 三、 特殊醉酒场景下的安全监护说明

为了保障生命与财产安全，e代驾明确规定：
* **醉酒车主必须有人陪伴**：若车主已处于失去意识的重度醉酒状态，**车内必须有清醒的同伴陪伴**。
* **司机有权拒绝无陪伴醉酒单**：为了防止在行车途中因醉酒客户呕吐、拉扯方向盘而引发交通事故，若无清醒人员陪同，代驾司机有权为了安全考量合理拒绝提供代驾服务。
